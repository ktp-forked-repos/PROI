Biblioteka operacji na macierzach i wektorach rzadkich.

Biblioteka umozliwia wykonywanie operacji na nastepujacych obiektach:
:: wektor pelny
:: wektor rzadki
:: macierz pelna
:: macierz rzadka A (wiersze macierzy sa wektorami rzadkimi)
:: macierz rzadka B (kolumny macierzy sa wektorami rzadkimi)
:: macierz rzadka C (elementy niezerowe sa w liscie)

Zdefiniowane operacje:
:: dodawanie wektorow
:: dodawanie macierzy
:: mnozenie macierzy
:: mnozenie macierzy przez wektor
:: liczenie sladu macierzy
:: transpozycja macierzy
:: wprowadzanie i wyprowadzanie obiektu do pliku
